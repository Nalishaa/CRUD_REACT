# Grocery Bud

A CRUD grocery list app built with React + Vite.

## Features

- Add, edit, and delete grocery items
- Check off items with a strikethrough
- Data persists with localStorage
- Toast notifications for actions

## Tech Stack

- React
- Vite
- nanoid
- react-toastify
